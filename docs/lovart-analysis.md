# Lovart Editor 设计分析（供 R9 UI 减重参考）

## 布局结构
- **两栏布局**：左=无限画布（tldraw），右=AI 对话面板
- **没有左侧面板**：没有 Layers、Pages、Assets、Templates、Components、History
- **没有属性面板**：选中元素不弹属性编辑器，所有修改通过 AI 对话

## 顶部导航
- 左上：logo 图标 + 可编辑文件名（极小字号）
- 右上：积分显示 + 用户头像 + "对话" 按钮
- 中间完全空白，不占空间

## 工具栏
- 浮在画布底部中央，一排 icon 按钮（约 8-9 个）
- 纯图标，没有文字标签
- 右下角有缩放控制（百分比 + 加减按钮）
- 工具栏背景半透明，不遮挡画布

## 右侧 AI 面板
- 顶部：prompt 输入区 + 历史对话按钮 + 附件按钮
- 中间：对话消息流（支持图片预览、工具调用展示）
- 底部：输入框 "Start with an idea, or type @ to mention" + Agent 模式切换
- 没有 tab 切换（没有 Create/Spec/Ship）
- 没有 Brand 设置、Provider 选择等配置项外露

## 视觉风格
- 深色主题为主
- 面板之间几乎没有 border，用微妙阴影分隔
- 圆角统一且克制
- 字号偏小，信息密度低
- 大量留白
